Class: upsertTEI

Adaptor.upsertTEI(uniqueAttributeId, data, optionsopt, callbackopt) → {Promise.<state>}

new upsertTEI(uniqueAttributeId, data, optionsopt, callbackopt) → {Promise.<state>}

Upsert(Create or update) one or many Tracked Entity Instances. Update if the record exists otherwise insert a new record. This is useful for idempotency and duplicate management
